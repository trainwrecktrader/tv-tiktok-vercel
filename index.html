TradingView → TikTok Bridge

This Vercel project exposes a webhook at:

/api/tradingview-webhook

Point your TradingView alert webhook URL at https://YOUR-VERCEL-URL/api/tradingview-webhook?secret=YOUR_SECRET.

When your Pine Script fires alert() with a JSON payload, this function will parse the liquidity levels and send them to TikTok using your configured credentials.
